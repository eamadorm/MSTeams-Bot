# Future Roadmap

## Phase 1: Foundation (Current)
*   Interactive Portfolio Dashboard.
*   Mock data integration with Gemini-powered search simulation.
*   Static "Document Viewer" simulation.
*   Basic industry-specific tracks.

## Phase 2: Intelligence & Integration (Q3 2024)
*   **Live PDF Processing:** Integration with Gemini 1.5/2.0 multimodal capabilities to ingest actual PDF files.
*   **Vector Search (RAG):** Implement a Vector Database (Pinecone) to enable Retrieval-Augmented Generation across the entire document corpus.
*   **Real-time Notifications:** Alerts for contracts expiring in < 30 days.

## Phase 3: Advanced Legal AI (Q4 2024)
*   **Redline Suggestions:** AI-driven suggestions for modifying high-risk clauses to match "Gold Standards".
*   **Legacy Remediation:** Batch processing tool to identify and update "LIBOR" or "GDPR" references across thousands of old contracts.
*   **Multi-language Support:** Support for extracting data from contracts in French, German, and Spanish.

## Phase 4: Enterprise Ecosystem (2025)
*   **API Connectors:** Direct integrations with Salesforce, DocuSign, and SAP.
*   **Audit Trails:** Blockchain-verified logs for contract changes and AI interpretations for regulatory compliance.
*   **Custom Model Fine-tuning:** Allow enterprises to fine-tune extraction models on their specific legal vernacular.
